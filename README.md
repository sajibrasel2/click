# সাইট ক্লিক ট্র্যাকার

এই প্রজেক্টটি ৫টি সাইটের বিজ্ঞাপন ক্লিক সংখ্যা ট্র্যাক করার জন্য তৈরি করা হয়েছে। XAMPP পরিবেশে এটি চালাতে পারবেন।

## ফাইলসমূহ

- `config.php` - ডাটাবেস সংযোগ সেটিংস
- `install.php` - ডাটাবেস, টেবিল এবং ডিফল্ট ব্যবহারকারী/সাইট তৈরি করে
- `index.php` - লগইন পেজ
- `dashboard.php` - সাইটের ক্লিক পরিসংখ্যান দেখায়
- `visit.php` - ক্লিক হিসাব করে এবং সাইটের বিজ্ঞাপন URL এ রিডাইরেক্ট করে
- `logout.php` - লগআউট করে
- `styles.css` - স্টাইল

## ইনস্টলেশন

1. আপনার XAMPP এর MySQL চালু করুন।
2. ফোল্ডারটি `c:/xampp/htdocs/click` এ রাখুন।
3. `config.php` এ আপনার MySQL সেটিংস দিন। সাধারণত:
   - host: `127.0.0.1`
   - database: `adsterra_clicks`
   - user: `root`
   - password: ` ``
4. ব্রাউজারে যান: `http://localhost/click/install.php`
5. ইনস্টলেশন সফল হলে `http://localhost/click/index.php` এ যান।

## ডিফল্ট লগইন

- ব্যবহারকারী: `user1`, `user2`, `user3`, `user4`, `user5`
- পাসওয়ার্ড: `pass1234`

## ব্যবহার

- লগইন করার পর `dashboard.php` এ ৫টি সাইট দেখাবে।
- "বিজ্ঞাপন পরিদর্শন" বাটনে ক্লিক করলে ক্লিক রেকর্ড হবে এবং আপনাকে সেই সাইটে পাঠাবে।
- প্রতিটি সাইটের আজকের ক্লিক এবং মোট ক্লিক দেখানো হবে।

## কাস্টমাইজেশন

- ৫টি সাইটের নাম ও URL পরিবর্তন করতে `sites` টেবিলে `ad_url` আপডেট করুন।
- আপনার Adsterra বিজ্ঞাপন URL এখানে সেট করুন যাতে ক্লিক রিডাইরেক্ট ঠিক কাজ করে।
- নতুন ব্যবহারকারী যোগ করতে `users` টেবিলে ইনসার্ট করুন।
